Course to Mindmap Studio

把视频课程内容，压成一张真正能复习的结构化脑图

这是一个为学习者设计的轻量 Web 原型：你贴入课程字幕稿、讲义或笔记，它会帮你整理出清晰的章节结构、 关键概念、方法步骤、案例提醒和行动项，同时给出一套可直接接入大模型的高质量 Prompt。

输入 课程字幕稿 / 学习笔记 / 讲义
输出 结构化脑图 + Markdown + JSON
适合 知识整理、课程复习、二次创作

Step 1

课程拆解输入

先描述课程上下文，再贴入原始内容。首版原型支持你手动粘贴视频转录文本、课程摘要或老师讲义。

01 去口语冗余

清理寒暄、重复和噪声，让课程主线先显形。

02 章节化切分

按主题或逻辑步骤拆出更容易复习的模块。

03 脑图化表达

围绕概念、方法、案例、误区、行动项组织层级。

04 Prompt 工作台

为后续接入模型保留结构化提示词模板。

导图结果

课程总览

贴入课程内容后，这里会生成一段简洁摘要，告诉你这门课的主线、适合谁，以及大致包含多少个模块。

无限画布脑图

空白处拖动画布，滚轮缩放，拖住节点右上角手柄移动，直接点击文字即可编辑。
